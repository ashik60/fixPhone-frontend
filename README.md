# FixPhone

View website live [here](https://fix-phone-6a509.web.app/).

## Features

-   Separate Admin and User Dashboard
-   Buy a srvice
-   Sttripe Payment gateway
-   User can post a review
-   Admin can
    -   Add service
    -   Change order status
    -   Delete a service
    -   Add another admin

## Technology used

-   React js
-   React Router DOM
-   Firebase Auth
-   Firebase Hosting
-   Bootstrap
-   React Bootstrap
-   React hook form
-   and much more...
